## Usage

first of all you need to install nodejs  
- [Nodejs Home](https://nodejs.org/en/ï¼‰

``` 
npm -version
```

```

```

```  
cmd

```
copy the .jdl and run the command

copy the OBDML02-auth.jh file into your project directory
``` 
luna genform OBDML02-auth.jh --force
```

``` 
mvnw
```
